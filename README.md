# Paper-pdf-renamer

一个用于批量整理学术 PDF 文件名的 Windows 桌面小工具。程序会读取 PDF 前几页文本，自动识别 DOI，并通过 CrossRef 获取论文标题、期刊、年份、接受日期、出版日期、第一作者等信息，然后按自定义规则生成新的文件名。
在获取到期刊名或 ISSN 后，程序会继续通过爱科学 SCI 期刊接口查询期刊影响因子、JCR 分区、中科院分区等信息，可直接加入文件名。

## 功能特点

- 批量扫描指定文件夹下的 PDF 文件
- 自动提取 DOI，并通过 CrossRef 获取论文元数据
- 自动查询爱科学 SCI 期刊信息，包括影响因子、JCR 分区、中科院分区、小类学科、出版社、OA 等
- 支持标题、期刊名、期刊缩写、ISSN、E-ISSN、年份、接受日期、出版日期、第一作者、DOI、影响因子、JCR、中科院分区等命名字段
- 支持在图形界面中自由调整字段顺序和分隔符
- 支持自由增加、减少、调整文件预览表中的信息列
- 支持点击文件预览表头按任意列升序/降序排序
- 支持将当前文件预览导出为 Excel `.xlsx`
- 重命名前可预览新文件名
- 自动避免覆盖同名文件
- 配置会保存到用户目录下的 `.pdf_renamer_config.json`

## 运行方式

### 方式一：直接运行打包好的程序

双击：

```text
PDF_Renamer.exe
```

### 方式二：用 Python 运行源码

先安装依赖：

```bash
pip install pypdf
```

然后运行：

```bash
python pdf_renamer.py
```

也可以直接双击：

```text
启动.bat
```

## 打包为 exe

在项目目录中双击：

```text
build.bat
```

或在命令行运行：

```bash
python -m pip install --upgrade pypdf pyinstaller
pyinstaller --onefile --windowed --noconfirm --clean --name PDF_Renamer pdf_renamer.py
```

打包完成后，生成的程序会复制到项目根目录：

```text
PDF_Renamer.exe
```

## 使用步骤

1. 打开程序。
2. 点击“选择文件夹”，选择存放 PDF 文献的文件夹。
3. 在“命名规则”区域选择需要的字段和分隔符。
4. 在“文件预览信息”区域选择需要显示在表格里的信息列。
5. 点击“扫描并提取”，等待程序读取论文元数据并查询期刊信息。
6. 检查“新文件名（预览）”是否符合预期。
7. 点击“执行重命名”。

## 文件预览信息

“命名规则”和“文件预览信息”互相独立：命名规则只决定最终文件名，文件预览信息只决定表格里显示哪些列。原文件名和新文件名固定显示，其余列可添加、删除、上移、下移或恢复默认。配置会随命名规则一起保存到用户目录下的 `.pdf_renamer_config.json`。

文件预览表支持排序：点击任意列的表头按该列升序排序，再次点击同一列表头切换为降序。表头上的箭头表示当前排序方向。

点击“导出Excel”可将当前预览表导出为 `.xlsx` 文件。导出的列顺序、显示字段和行顺序与界面中的当前预览保持一致。

## 期刊信息字段

期刊信息来自爱科学 SCI 期刊搜索接口，查询时优先使用 ISSN/E-ISSN，缺失时使用期刊名匹配。可加入命名规则的新增字段包括：

- `Impact-Factor  影响因子`
- `IF-Year  影响因子年份`
- `JCR  分区`
- `CAS  中科院分区`
- `CAS-Category  中科院大类`
- `CAS-Quartile  中科院区`
- `Subject  小类学科`
- `Publisher  出版商`
- `5-Year-IF  五年影响因子`
- `OA  开放访问`

爱科学为第三方数据源，正式投稿、考核或归档前建议再用 Clarivate JCR 或中科院分区表复核。
